# 🫀 Kalp Krizi Risk Tahmini

Hugging Face tabanlı makine öğrenmesi modeliyle kalp krizi risk skoru hesaplayan Flutter uygulaması.

## 🚀 Özellikler
- 🤖 Hugging Face üzerinde yayımlanan ML modeli ile risk tahmini
- 📊 Karar Ağacı ve Rastgele Orman algoritmaları
- 📱 Flutter ile cross-platform mobil arayüz
- 🔗 Python (Flask/FastAPI) tabanlı backend API entegrasyonu
- 📋 Kaggle veri seti ile eğitilmiş model

## 🛠️ Kullanılan Teknolojiler
- **Flutter** — Mobil uygulama
- **Python (Flask/FastAPI)** — Backend API
- **Hugging Face** — Model yayımlama
- **scikit-learn** — Model eğitimi
- **Kaggle** — Veri seti

## ⚙️ Kurulum
```bash
git clone https://github.com/Umut8001/kalp-krizi-risk-tahmini.git
cd kalp-krizi-risk-tahmini
flutter pub get
flutter run
```
